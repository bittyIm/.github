## Funtion

- Authorization
   - Login
   - Register
   - Reset password
- User
   - User profile
   - User settings
- Session
   - Hang
   - Reply
   - Delete
- Contacts
   - Contact list
   - Search for contacts
   - Add contacts
- Group
   - Create groups
   - Invite friends
   - Group member settings
   - Search for groups
   - Add groups
- Information
   - Apply to be added as a friend
   - Agree to be added as friend
   - Apply to join group chat
   - Agree to join the group chat
   - Invite to join group chat
- I18n
   - en 
   - cn
- Message
   - Markdown
   - Image
   - File
   - Video
   - Audio
